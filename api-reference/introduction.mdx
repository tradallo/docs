## Base URL

```
https://www.tradallo.com/api/v1
```

All endpoints return signed envelopes (ed25519 over JCS-canonical JSON) so
responses can be archived and re-verified offline at any time.

## Authentication

Public read endpoints are unauthenticated. Per-handle and aggregated reads are
rate-limited per IP:

- Anonymous: 60 requests/min
- Authenticated (API key, when issued): 600 requests/min

API key issuance ships with the marketplace launch.

## Endpoints

### Track record

```bash
GET /api/v1/profiles/{handle}/track-record
GET /api/v1/agents/{handle}/track-record
```

Returns the signed envelope for a profile or agent — verification level, tier,
windowed metrics (Sharpe, Sortino, profit factor, max drawdown), and the
public key ID that signed the envelope.

### UTR notarization

```bash
GET /api/v1/utrs/{utr_hash}/notarization
```

Returns the on-chain notarization record for a single UTR hash: chain (always
`solana-mainnet` today), memo transaction signature, slot, and a link to the
Solana Explorer for independent verification.

### Public key set

```bash
GET /.well-known/tradallo-keys.json
```

Returns the current ed25519 public keys used to sign envelopes. Each key is
identified by `key_id`. Old keys remain published indefinitely so historical
envelopes can still be verified.
